## Installing Keyboard Layout Emulation
### From Blender's official repository
> At the moment the extension is being reviewed for acceptance within Blender's extension repository.
> See how to install [from GitHub](#from-github)

1. Navigate to `Preferences > Get Extensions`:
    1. Open Blender preferences (`Edit > Preferences...` or `Ctrl + ,`)
    2. Select the `Get Extensions` tab on the left column.
2. In the top search bar, type `Keyboard Layout Emulation`
3. Find the extension in the list below and click `Install`
    1. If the extension entry appears in the *Installed* list, you're done.
    2. Otherwise, look for it in the *Available* list.

### From GitHub
1. Download the latest `add-on-keyboard-layout-emulation-v*.*.*.zip` from
   [GitHub](https://github.com/endorh/blender-keyboard-layout-emulation/releases)
2. In Blender, navigate to `Preferences > Get Extensions`
3. In the top right corner, click the dropdown and select `Install from Disk...`
4. Select the downloaded `zip` file and confirm `Install from Disk`.

> Once installed, you should be able to find a `Keyboard Layout Emulation` entry in
> the `Preferences > Get Extensions` and `Preferences > Add-ons` menus.

### Uninstalling
Disabling and/or uninstalling the extension will automatically revert keyboard
layout emulation to avoid damaging your keymap.
If you want to preserve the emulation after disabling/uninstalling the extension,
you may configure the [uninstall options](preferences.md#uninstall-options)
before disabling/uninstalling the extension.

1. Navigate to `Preferences > Get Extensions`
2. Type `Keyboard Layout Emulation` in the search bar
3. If you find the entry in the *Installed* list below, click its
   dropdown button on its right and select `Uninstall`
